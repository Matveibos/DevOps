Задание 1

1)Для того чтобы "подбросить" файл в контейнер необходимо создать том (он же volume).
1.1) Необходимо выполнить команду docker volume create --name <ИМЯ> я назвал том <my_volume>
2)После создания тома можно посмотреть точку монтирования где лежат его файлы.
2.1) Необходимо выполнить команду docker volume inspect my_volume, в моем случае путь был таков "Mountpoint": "/var/lib/docker/volumes/my_volume/_data"
3)Необходимо перейти по этому пути и создать там необходимые файлы, в моем случа файл index.html.(файл index.html лежит в папке с домашним)
4)Необходимо запустить контейнер с флагом --mount, таким образом мы "примонтируем" данный volume к контейнеру по указанному пути.
4.1) Необходимо выполнить команду  docker container run --mount source=my_volume,target=/usr/share/nginx/html -d -p 8080:80 nginx

2)В файле нужно прописать дополнительно параметр volume (этот параметр как раз даёт возможноть "подбросить" файлы по указанному пути)
